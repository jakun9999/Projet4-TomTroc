# Projet4-TomTroc

Repository project 4 - Openclassrooms Full stack developper training.

## Technologies

- HTML/CSS
- TAILWINDCSS 4.3
- DOCKER
- COMPOSER (autoloader)
- MYSQL

## How to install

```
npm uninstall tailwindcss @tailwindcss/cli @tailwindcss/oxide-linux-x64-gnu

```

## How to start dev mode

```
docker compose up --build -d
npm run dev:css

```

## How to run production mode

```
Description starting production mode (without PHPMYADMIN)
```
